# installation en local
- necessite nodejs, npm, bower

```
git clone https://github.com/scenaristeur/Pa-Comp.git Pa-Comp
cd Pa-Comp
npm install
cd public
bower install
cd ..
node . 
```

Le serveur doit être accessible à l'adresse http://127.0.0.1:3000
